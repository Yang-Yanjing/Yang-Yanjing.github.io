## Biography

I am a direct-admission Ph.D. student in Software Engineering at the State Key Laboratory of New Technology of Computer Software, Nanjing University. My research focuses on source code security and API security, especially large language model based vulnerability detection, robustness and generalization of vulnerability detection models, and agentic AI for API security analysis.

In software security, I study how LLM agents can support vulnerability detection and how vulnerability reports drive code changes across the software supply chain. In API security, I work on agent-based analysis of API entry points, calling processes, malicious payloads, abnormal API call patterns, and broken access control risks.

## Research Projects

### Multi-Agent Collaboration Task Error Detection

CCF Populus Grove Project, Sep. 2025 - Present  
We study anomaly detection in multi-agent collaboration systems during code generation. The project aims to identify, correct, and feed back task-level errors to improve the reliability of multi-agent code generation.

### Vulnerability Detection Based on LLM in OpenHarmony

CCF Populus Grove Project, Sep. 2024 - Aug. 2025  
We fine-tuned LLMs with open-source HarmonyOS vulnerability information to improve vulnerability detection. I was responsible for detection model design and optimization, building on the DLAP framework and adapting it to enterprise requirements in the HarmonyOS ecosystem.

### PaaS API Security Guardrail

CCF Populus Grove Project, Mar. 2024 - Aug. 2024  
We used agentic AI to enhance API security by detecting malicious payloads and abnormal API call patterns. I designed the core algorithms, optimized key metrics, and served as the first author for the results. The work produced a black-box API discovery algorithm and methods for payload injection and broken access control risks from the OWASP API Security Top 10. The solutions were deployed in a Cloud PaaS intelligent gateway and led to two enterprise patents.

## Work In Progress

### Towards Robust Vulnerability Detection: Agentic-AI-Constructed Meta Data for Meta Self-Paced Noise-Robust Learning

This research explores how to teach AI to detect software vulnerabilities in a way that mirrors human learning: starting from simpler, more reliable examples and gradually moving to harder ones. We use agentic AI to build a clean and trustworthy meta dataset, which guides a meta self-paced learning process and improves robustness to noisy real-world labels.
